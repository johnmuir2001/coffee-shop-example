# Coffee Shop Instructions

To get this code and to get it to run in the terminal first you need to clone the repo. In your terminal, navigate to where you want this project folder to be located and then type the command:
```
git clone https://github.com/johnmuir2001/coffee-shop-example.git
```

You will then need to install the packages used in this project. Navigate to the newly created project folder and run this command:

```
npm install
```

This command will read through the package.json and install any packages located in the dependencies section. You will then be able to run the project by running the JS file coffeeShop.js.
